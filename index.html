Entendendo comunicação Client X Server

(Voltar)

Nesta aula, o professor explicou os fundamentos da Internet.
Os fundamentos apresentados foram:

    A história da Internet;
    Joseph Carl plantou a semente da internet ao teorizar
    sobre uma rede onde todos poderiam se comunicar através dela.
    A partir da semente plantada, muitas pessoas e instituições se dedicaram
    ao desenvolvimento da internet. Desde a Arpanet, Grace Hoppe, Tim Berners-Lee,
    passando pelo Bill Gates e até o Steve Jobs.
    Cada um deles tiveram papel fundamental para que hoje possamos
    interagir, consumir, trabalhar e até nos entretermos através da internet.
    Clients e Servers;
    O Client faz uma requisição, em HTTP, para a internet. A internet,
    vai até um Server e retorna com a requisição inicial.
    Linguagens de programação.

    Neste contexto, o professor apresenta as linguagens de
    programação nos dois lados, Server e Client.

    Alguns exemplos de linguagens são:
    Client: Java Script;
    Server: C#, Ruby, PHP, etc

Introdução ao HTML na pratica

(Voltar)

Nesta aula, o professor explicou os conceitos iniciais do HTMl.
Os conceitos iniciais apresentados foram:

    Apresentação do curso, das ferramentas e do Inspetor de elementos;

    Nesta aula o professor apresentou-se, detalhou o objetivo do curso,
    e deu algumas informações para iniciarmos nossa jornada nesse curso.
    Como por exemplo: Instalar o VS Code, usar o inspetor de elementos, etc.

    A estrutura básica do HTML;
    html
    head
    /head
    body
    /body
    /html
    * as Tags estão dentro dos sinais de maior e menor.
    * / indica o fechamento da tag.

    HTML:Indica que é uma linguagem de marcação de texto;
    HEAD:Indica que o que está dentro desta tag será como uma "configuração",
    Isto é, não ficará visível na página web, apenas no código.
    BODY:Indica a configuração visual na página web


    As Tags;

    As Tags podem ser definidas como uma marca que define o conteúdo de
    um documento da respectiva linguagem.

    Os atributos para as Tags;

    Define o estilo que queremos apresentar nossas Tags.
    Como por exemplo: Cor, Tamanho, etc.

    Textos;

    Define as Tags para criar textos e a tipografia do mesmo;
    H1(Título),H2(Subtitulo),...,H6,p(Paragrafo) e Blockquote (Citação).

    As listas ordenadas e não ordenadas;
    Define as Tags para criamos lista ordenada (ol) e não ordenada(ul).
    Independente das listas, o il deve estar dentro delas.
    Sendo o(il) um Item da Lista.

    Os links.

    Define as Tags para incluir Links em sua pagina,
    podendo ser abertas na propria pagina ou em outra.
